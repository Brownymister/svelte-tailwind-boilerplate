Biolerplate project for using tailwind within svelte.

## Quick start 

1. Make sure that you have node and npm installed.
2. Cloe this Repo like this: `git clone --depth=1 https://github.com/Brownymister/svelte-tailwind-boilerplate.git <YOUR_PROJECT_NAME>`
3. Change into the project directory: `cd <YOUR_PROJECT_NAME>`
4. Run `npm install` for installing Dependencies 
5. Run `npm run dev` for starting the dev server.
